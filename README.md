#  Restaurant Landing Page


##  Overview

This landing page is designed to give a restaurant a strong online presence by presenting its brand, menu, ambience, and contact details in an attractive, easy-to-navigate single-page layout. It focuses on visual appeal, clear information hierarchy, and a smooth user experience to encourage visitors to explore the menu and make a reservation.

---

##  Features

- **Hero Section** with an eye-catching banner, restaurant name, tagline, and call-to-action button
- **About/Story Section** describing the restaurant's background and philosophy
- **Menu Section** showcasing dishes with images, descriptions, and prices
- **Gallery Section** displaying high-quality food and ambience photos
- **Testimonials Section** featuring customer reviews and ratings
- **Reservation/Contact Section** with a booking form and location details
- **Sticky navigation bar** with smooth scrolling to each section
- **Fully responsive design** for mobile, tablet, and desktop devices
- **Interactive elements** such as hover effects, image sliders, and animations

---

##  Technologies Used

- **HTML5** – Semantic structure and content
- **CSS3** – Styling, layout (Flexbox/Grid), and animations
- **JavaScript** – Navigation toggle, smooth scrolling, sliders, and form interactions

#  Temperature Converter

A simple and responsive web application that converts temperatures between **Celsius**, **Fahrenheit**, and **Kelvin**. This project was built as part of my **Web Development and Designing internship at Oasis Infobyte**.


##  Project Overview

The Temperature Converter is a lightweight, browser-based tool that allows users to instantly convert temperature values between different units. It demonstrates core front-end development concepts including DOM manipulation, event handling, and responsive UI design.



##  Features

- Convert temperature between **Celsius, Fahrenheit, and Kelvin**
- Real-time, instant conversion on user input
- Clean, minimal, and responsive user interface
- Input validation to handle invalid or empty entries
- Works seamlessly across desktop and mobile devices

---

##  How It Works

The converter takes a numeric input value along with the selected unit and applies the standard temperature conversion formulas:

| Conversion | Formula |
|---|---|
| Celsius → Fahrenheit | `(C × 9/5) + 32` |
| Fahrenheit → Celsius | `(F − 32) × 5/9` |
| Celsius → Kelvin | `C + 273.15` |
| Kelvin → Celsius | `K − 273.15` |
| Fahrenheit → Kelvin | `(F − 32) × 5/9 + 273.15` |
| Kelvin → Fahrenheit | `(K − 273.15) × 9/5 + 32` |

JavaScript's `addEventListener()` is used to detect user input or button clicks, after which the appropriate formula is applied and the result is dynamically displayed on the page using DOM manipulation (`innerHTML` / `textContent`).
